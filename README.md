# Codex `/pet`：刻晴

这是可由 Codex `/pet` 功能加载的刻晴人形 SD 动画角色包，不是动物化宠物。

本目录同时是一个独立 Git 仓库；成品图集、配置、动画预览与验收资料均纳入版本控制。

## 安装位置

```text
~/.codex/pets/keqing/
├── pet.json
└── spritesheet.webp
```

本次构建已经安装到上述目录。

## 图集契约

- 图集尺寸：1536 × 1872
- 网格：8 列 × 9 行
- 单格：192 × 208
- 格式：带透明通道的 WebP
- 未使用格：完全透明

## 动画状态

1. `idle`：呼吸、眨眼和轻微身体起伏
2. `running-right`：向右疾跑
3. `running-left`：向左疾跑；单独生成以保留刻晴左右不对称的发饰与服装细节
4. `waving`：克制的招手问候
5. `jumping`：蓄力、起跳、最高点、下降、站稳
6. `failed`：保持佩剑可见的克制失落反应
7. `waiting`：等待用户指示的询问姿态
8. `running`：双脚落地的专注任务执行状态，不是原地跑步
9. `review`：通过眼神、前倾和持剑变化表现审查

## 角色生成约束

```text
角色必须是《原神》的刻晴本人，而不是动物、灵兽或拟兽。使用适合 Codex /pet
小尺寸显示的人形 SD 比例，同时忠实保留紫罗兰长发双马尾、猫耳状发髻、紫晶眼、
紫白金原版服装、花形金饰、长手套、带金纹的深色连裤袜、紫色鞋和纤细单手剑。
禁止真实兽耳、动物尾巴、兽爪、服装改编、其他角色、文字、场景、阴影和游离特效。
所有动画帧保持同一张脸、发型、服装、佩剑、身体比例与左右不对称细节。
```
